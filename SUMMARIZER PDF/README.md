AI Summary GPT Bot is a powerful Telegram bot that uses modern Large Language Models (LLMs) to automatically generate concise summaries from different types of content. The bot can summarize plain text, web articles, PDFs, and even YouTube videos, helping users quickly understand key information without reading the entire content. �

✨ Key Features
📄 Text Summarization – Convert long text into short, meaningful summaries.
🌐 URL Summarization – Extract and summarize content directly from web links.
📑 PDF Support – Upload documents and get quick summaries instantly.
🎥 YouTube Video Summaries – Understand video content without watching the entire video.
🤖 AI Powered – Uses advanced GPT-based models for accurate and intelligent summarization.
🔐 Private Usage – Configure allowed users to restrict bot access.
⚡ Fast & Efficient – Handles large inputs using chunk processing.
